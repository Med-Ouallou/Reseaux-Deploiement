Qu'est-ce que Ubuntu ? 🐧
Un système d'exploitation open-source basé sur Linux

Ubuntu est une distribution Linux — un système d'exploitation complet bâti autour du noyau Linux. Il est totalement open-source et gratuit, développé et sponsorisé par l'entreprise Canonical depuis 2004.

🏢
Canonical Ltd.
Fondée par Mark Shuttleworth en 2004. Elle finance le développement, assure le support commercial et publie chaque nouvelle version.
🔱
Basé sur Debian
Ubuntu hérite du format de paquets .deb, des dépôts APT et de la stabilité légendaire de Debian GNU/Linux.
🌍
Utilisé partout
PC personnels, entreprises, serveurs, cloud (AWS, GCP, Azure), IoT, et même la Station Spatiale Internationale !
🏆
Le plus populaire
Distro Linux n°1 sur desktop et serveurs cloud. La communauté répond à presque toutes les questions en ligne.

🧠 Le Kernel Linux
Cœur du système. Il gère la mémoire, les processus, la communication avec le matériel et la sécurité. Sans lui, rien ne fonctionne.

03 / 08
Le Système de Fichiers 📂
Fini le C:\ — sur Linux tout part de la racine /

Sur Windows tu as C:\, D:\… Sur Linux, il n'y a qu'un seul point de départ : la Racine notée / (root). Tout le système part de là — disques, périphériques USB, mémoire, tout.

/ ← La Racine (Root), point de départ de TOUT ├── /home ← Vos fichiers personnels (ex: /home/ali/Documents) ├── /etc ← Fichiers de configuration système (réseau, SSH…) ├── /var ← Fichiers variables : logs, bases de données, caches ├── /bin ← Commandes de base : ls, cp, mv, cat, bash… ├── /sbin ← Commandes admin (root) : fdisk, ifconfig… ├── /usr ← Applications installées & librairies utilisateur ├── /boot ← Noyau Linux + chargeur GRUB (démarrage) ├── /dev ← Périphériques : /dev/sda (disque), /dev/tty… ├── /proc ← Infos processus en temps réel (virtuel, en RAM) ├── /tmp ← Fichiers temporaires, vidé à chaque redémarrage └── /root ← Dossier personnel du super-utilisateur (root)
04 / 08
Gestion des Logiciels (APT) 📦
Pas de fichiers .exe — on passe par un gestionnaire de paquets

Sur Ubuntu, on n'installe pas les logiciels avec des fichiers .exe comme sur Windows. On utilise APT (Advanced Package Tool) qui télécharge depuis des dépôts officiels sécurisés. La commande sudo donne temporairement les droits d'administrateur.

ubuntu@pc:~$
$ sudo apt update           # Mettre à jour la liste des paquets
$ sudo apt upgrade          # Installer toutes les mises à jour
$ sudo apt install firefox    # Installer le navigateur Firefox
$ sudo apt remove firefox     # Désinstaller Firefox
$ apt search vlc           # Chercher un paquet par nom
$ apt show nodejs          # Afficher les infos d'un paquet
$ dpkg -l                 # Lister tous les paquets installés

Les Différentes Saveurs (Flavors) 🎨
Même moteur Ubuntu, différentes interfaces graphiques

Avantages & Inconvénients ⚖️
Une analyse honnête et complète
✅ Avantages
  • 🔒 Sécurité : très peu de virus, pas besoin d'antivirus
  • ⚡ Performances : fluide même sur de vieilles machines
  • 💰 Gratuit & Open Source : 100% libre
  • 🧑‍💻 Paradis des devs : Docker, Git, Python, Node natifs
  • ☁️ Cloud : standard sur AWS, GCP, Azure
  • 🔄 Mises à jour : système entier en une commande
  • 🌍 Communauté énorme : réponse à presque tout
  • 🎮 Gaming en progrès : Steam + Proton de mieux en mieux
❌ Inconvénients
  • 🎨 Suite Adobe : Photoshop, Premiere… absents
  • 📊 Microsoft Office : LibreOffice comme alternative
  • 🎮 Jeux AAA : moins de titres natifs (en amélioration)
  • 📚 Apprentissage : le terminal demande adaptation
  • 🔧 Drivers : certains matériels récents mal supportés
  • 📦 Snap : critiqué pour le démarrage lent des apps
  • 🏢 Canonical : décisions parfois impopulaires

Debian — La Mère d'Ubuntu 🔴
Le projet GNU/Linux le plus ancien, le plus stable, le plus influent
Debian, créé en 1993 par Ian Murdock, est le fondement sur lequel Ubuntu est bâti. Ubuntu utilise ses paquets .deb, son gestionnaire APT et sa philosophie de stabilité — en y ajoutant une interface soignée, un support commercial et un rythme de publication régulier.
📅
Fondé en 1993
Plus de 30 ans d'existence. L'une des distros les plus anciennes encore activement maintenues et utilisées en production.
🌊
3 branches
Stable (ultra-sûr, entreprises), Testing (intermédiaire), Unstable/Sid (bleeding edge, pour développeurs).
👥
100% Communauté
Dirigé uniquement par des bénévoles. Aucune entreprise derrière. Gouvernance démocratique avec vote des développeurs officiels.
🧩
Influence massive
Ubuntu, Linux Mint, Kali Linux, MX Linux, Raspbian (Raspberry Pi)… tous sont basés sur Debian.
